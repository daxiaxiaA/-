# 功能介绍

## 项目概述

数字签名与加密传输软件用于演示和验证“发送方签名并加密、接收方解密并验签”的完整流程。程序按照 `task.png` 中的流程设计：发送方 A 对明文 `M` 计算 Hash，使用 A 的私钥 `RK_A` 形成数字签名，再将 `M || 签名` 使用对称密钥 `K` 加密，并使用接收方 B 的公钥 `UK_B` 加密 `K`；接收方 B 使用私钥 `RK_B` 解出 `K`，还原 `M || 签名`，再使用 A 的公钥 `UK_A` 验证签名。

## 独立功能

1. 对称加密功能：支持 AES 和 DES，用户可以在界面中切换算法。
2. Hash 摘要功能：支持 SHA-256、SHA-1 和 MD5，用户可以在界面中切换算法。
3. RSA 密钥生成功能：可以生成发送方 A 和接收方 B 两组不同的公私钥。
4. 数字签名功能：使用发送方 A 的私钥对明文摘要加密形成签名。
5. 密钥封装功能：使用接收方 B 的公钥加密对称密钥 `K`。
6. 解密验签功能：接收方解出明文后重新计算 Hash，并和签名中的 Hash 比较。
7. 字符串输入功能：可直接在界面文本框中输入待处理内容。
8. 文件输入功能：可选择本地文件，对文件字节内容进行签名、加密、解密和验签。
9. 密文包保存与读取功能：可保存 `.crypto` 文件，并重新打开完成验证。
10. 命令行自检功能：可通过 `--self-test` 自动验证多组算法组合。

## 用户交互

程序提供 Swing 可视化界面，包含参数选择、明文输入、密钥信息、过程日志和解密结果区域。用户可以通过按钮完成生成密钥、执行完整流程、保存密文包、打开密文包和保存解密结果等操作。

## 可执行软件

项目可编译为可执行 jar：

```text
release/CryptoTaskApp.jar
```

运行命令：

```bash
java -jar release/CryptoTaskApp.jar
```
